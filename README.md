# OGQ 스티커 심리테스트 🌸

귀여운 OGQ 스티커로 알아보는 나의 성격! 48가지 개성 넘치는 캐릭터 중 나와 꼭 맞는 스티커를 찾아보세요.

## ✨ 주요 기능

- 🌺 **3가지 시작 경로**: 꽃, 나무, 해변 중 선택하여 시작
- 🎯 **48가지 결과**: 각자의 개성을 담은 다양한 OGQ 스티커 캐릭터
- 📱 **모바일 최적화**: 모바일 환경에 최적화된 반응형 디자인
- 📸 **인스타그램 스토리 공유**: 결과를 인스타그램 스토리에 바로 공유
- 🛒 **OGQ 마켓 연동**: 마음에 드는 스티커는 바로 구매 가능

## 🎨 디자인 특징

- 귀엽고 발랄한 한글 폰트 (Jua, Noto Sans KR)
- 파스텔 그라데이션 배경
- 부드러운 애니메이션 효과
- 터치 친화적인 큰 버튼
- SNS 공유에 최적화된 결과 화면

## 🛠️ 기술 스택

### Frontend
- **React 18** + **TypeScript**
- **Vite** - 빠른 개발 환경
- **Wouter** - 경량 라우팅
- **Tailwind CSS** - 유틸리티 기반 스타일링
- **Shadcn/ui** - 재사용 가능한 UI 컴포넌트
- **html2canvas** - 인스타그램 스토리 이미지 생성

### Backend
- **Express.js** - Node.js 웹 프레임워크
- **Drizzle ORM** - TypeScript ORM
- **PostgreSQL (Neon)** - 서버리스 데이터베이스

## 🚀 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5000` 접속

### 빌드

```bash
npm run build
```

## 📁 프로젝트 구조

```
ogq-sticker-quiz/
├── client/              # 프론트엔드 React 앱
│   ├── src/
│   │   ├── pages/      # 페이지 컴포넌트 (Home, Quiz, Result)
│   │   ├── components/ # UI 컴포넌트
│   │   └── lib/        # 유틸리티 함수
│   └── public/         # 정적 파일 (이미지 등)
├── server/             # 백엔드 Express 서버
├── shared/             # 프론트/백엔드 공유 타입 및 데이터
│   └── quizData.ts    # 퀴즈 질문 및 결과 데이터
└── scripts/            # 빌드/배포 스크립트
```

## 🎮 사용 방법

1. **시작 화면**에서 "시작하기" 버튼 클릭
2. **첫 번째 질문**에서 꽃🌸, 나무🌳, 해변🏖️ 중 하나 선택
3. 이어지는 **2-4개의 질문**에 답변
4. **결과 화면**에서 나와 어울리는 OGQ 스티커 확인
5. 마음에 들면 **인스타그램 스토리로 공유**하거나 **OGQ 마켓에서 구매**

## 📱 인스타그램 공유 기능

- 모바일: 자동으로 이미지 다운로드 후 인스타그램 앱 실행
- 데스크톱: 이미지 다운로드 안내 메시지 표시
- 1080x1920 (9:16) 인스타그램 스토리 최적화 이미지

## 🎯 퀴즈 구조

- **시작 질문**: 3가지 선택지 (각각 다른 경로로 분기)
- **후속 질문**: 경로별 2-4개의 질문
- **결과**: 48개의 고유한 OGQ 스티커 캐릭터
